## Описание эндпоинтов

Post запрос "/task" с body(title,description) добавляет новую задачу в очередь и возвращает стату добавления(true/false);

Post запрос "/data" без параметров, загружает первые задачи в бд;

Get запрос "/all" без  параметров, возвращает все загруженные задачи в бд;

Get запрос "/task" с query параметром (id) возвращает задачу по этому id;

Post запрос "/task_worker" с body (workerRq(name,position,avatar),idForTask) добавляет к задаче работника по переданому id и возвращает статус;

Put запрос "/task" с body(id,title,desckription,time,status) изменяет задачу по переданному id и возвращает статус;

Get запрос "/task_worker" с query парметром(id) возвращает задачу с работником по переданному id;

Get запрос "/worker" с query параметром(id) возвращает работника по переданному id;

Delete запрос "/worker" с query запросом(id) удаляет работника по переданному id и возвращает сататус;

Put запрос "/worker" с body(id,name,position,avatar) изменяет парметры работника по переданному id и возвращает статус.
